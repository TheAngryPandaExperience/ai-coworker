# AI Coworker

Autonomous AI software-development coworker environment for Cursor.

## Quick start

```powershell
npm install
.\start_local.ps1
```

Open http://localhost:3000

## Build a feature (existing project)

In a new Cursor chat:

```
Follow the protocol.

My idea is:
[your idea here]
```

See `AGENTS.md` for the full agent operating protocol.

## Start a new project

See **`NEW_PROJECT_GUIDE.md`** for copying this setup to a new app or bootstrapping a different stack.
